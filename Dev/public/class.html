TITULO DE LA CLASE DE TERAPIAS NATURALES

La tradición sugiere que el primer programa en un nuevo lenguaje debe imprimir las palabras “Hello, world” en la pantalla. En Swift, esto puede ser echo en una sola línea: printIn(“Hello, world”) Si tu has escrito código en C o Objective-C, esta sintaxis te va a ser familiar — en Swift, esta linea de código es un programa completo. Tu no necesitas de importar una librería separada para funcionalidad como entrada/salida o manejo de cadenas de texto. Código escrito en el ámbito global es utilizado como el punto de entrada para el programa, entonces tu no necesitas una función principal (main). Tu tampoco necesitas escribir puntos y comas al final de cada declaración. Este tour te proporciona la información necesaria para empezar a escribir código en Swift por enseñandote como lograr una variedad en tareas de programación. No te preocupes si tu no entiendes algo — Todo lo introducido en este tour esta expandido en detalle en el resto de este libro. NOTE Para la mejor experiencia, abre este episodio como un playground en Xcode. Playgrounds te permite editar los listados de código y ver el resultado inmediatamente.
